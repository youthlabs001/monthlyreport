# Monthly Report

고객 매출 현황을 확인할 수 있는 월간 리포트 시스템입니다.

## 기능

- 이메일/비밀번호 로그인
- 월별 매출 현황 조회
- 매출 통계 대시보드

## 기술 스택

- HTML, CSS, JavaScript
- Supabase (인증 & 데이터베이스)
- Vercel (배포)

## 설정

1. Supabase 프로젝트 생성
2. `js/config.js`에 Supabase URL과 anon key 설정
3. Supabase SQL Editor에서 `database/schema.sql` 실행

## 배포

Vercel에 자동 배포됩니다.
